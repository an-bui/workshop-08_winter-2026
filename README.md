# ENVS-193DS_workshop-08

## General information

This repo is for demonstrating model selection using AIC.  

This is a repo forked from the ENVS-193DS-2026 organization. I am the sole contributor.

## Data and file information

The `data` folder includes `Valliere_etal_EcoApps_Data.xlsx`, which is from: Valliere, Justin; Zhang, Jacqueline; Sharifi, M.; Rundel, Philip (2019). Data from: Can we condition native plants to increase drought tolerance and improve restoration success? [Dataset]. Dryad. [https://doi.org/10.5061/dryad.v0861f7](https://doi.org/10.5061/dryad.v0861f7)  

The `code` folder includes `workshop-code.qmd`, which contains all code for fitting and comparing models.
